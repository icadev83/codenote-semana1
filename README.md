---Descripción del proyecto---
Este proyecto es una pequeña aplicación web hecha en PHP para registrar calzado deportivo mediante un formulario. El usuario puede introducir datos como referencia, modelo, talla, color y precio, y el sistema se encarga de validarlos y mostrar un mensaje de confirmación.

---Funcionalidades---
Formulario sencillo e intuitivo
Validación de campos obligatorios
Comprobación de valores numéricos (precio, talla y referencia)
Mensajes dinámicos de éxito o error
Uso de sesiones para evitar envíos duplicados
Diseño básico con HTML y CSS

---Cómo funciona---
El usuario completa el formulario y, al enviarlo, los datos se limpian y validan. Si todo es correcto, se guarda un mensaje en sesión y se redirige a la misma página para evitar duplicaciones. Finalmente, se muestra el resultado al usuario.

---Tecnologías---
PHP · HTML5 · CSS3 · Sesiones en PHP

Interfaz sencilla y centrada, pensada para una experiencia clara y directa.
